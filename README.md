# Formatizer

## Installation
  - With **Yarn**, `yarn add formatizer`
  - With **NPM**, `npm install formatizer`

## Synopsis

## Code Example
```js
/*
 * Package Import
 */
import Formatizer from 'formatizer';

// ...

/*
 * Code
 */
const message = '*test*';


/*
 * Component
 */
const App = () => (
  <div>
    <Formatizer messange={message} />
    // Output: test in bold
  </div>
)
```
